github搭建博客小结

Aug 25, 2019点击

使用github+hexo搭建个人博客
1.申请github账号，创建仓库
2.安装node.js
3.安装git

4.githubssh配置
5.hexo安装：
在blog安装hexo:npm install -g hexo-cli
初始化:hexo init lzcxfz（创建博客根目录）
进入博客目录:cd lzcxfz
安装:npm install
本地测试:hexo server 打开localhost:4000
6.提交到github
hexo g生成 hexo d
如果提示：Deployer not found: git
就安装:npm install –save hexo-deployer-git
