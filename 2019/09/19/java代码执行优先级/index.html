java代码执行优先级

Sep 19, 2019点击

对象属性的初始化有三种方式
故意把初始化块，放在构造方法下面，问题：
这三种方式，谁先执行？谁后执行

答案:
一 类 属性
二 静态代码块
三 代码块
四 构造方法代码块


按照以上执行顺序，先执行public static int y = A.x + 1;此时y=0,载入A，x=0+1=1，再回到y = A.x + 1，y=1+1=2
静态属性和静态代码块是按顺序执行，哪个在前面就先执行哪个，非静态变量和方法都是要new的
